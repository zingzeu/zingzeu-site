安装

真鸟囝输入法正处于内测阶段，不稳定，暂时不推荐入门用户使用。你可以先试用网页体验版。

要安装真鸟囝输入法，请先到 下载 页面下载安装包。

  1. 双击运行你下载到安装包。

  2. Windows Defender 可能会警告你。

    此时可以点击“更多信息” -> "仍要运行"。

    已知问题: Windows 10 自带的防毒软件会将真鸟囝输入法误认做病毒。你可能需要暂时禁用 Windows Defender 或其他杀毒软件。

  3. Windows 用户账户控制 可能会提示你是否授权。此时选择“是”。（安装输入法需要系统管理员权限）

  4. 此时应该看到真鸟囝输入法安装器的界面。点击相应版本旁的“安装”按钮即可安装。

  5. 安装完成后，点击系统状态栏上的输入法切换按钮。

    选择“真鸟囝输入法”。

  6. 恭喜你，你可以开始试用真鸟囝输入法了。

下一步：阅读 榕拼规范白皮书 了解如何使用真鸟囝输入法打福州话。
